# Добавление позиции через мониторинг {#add_position}

:::info
Добавлять позиции могут только пользователи с правом доступа «Добавлять и редактировать позиции». Подробнее о правах доступа см. в разделе [Права доступа](../account_management/access_rights.mdx).
:::

Позиции можно добавлять вручную через административную панель на страницах мониторинга. Чтобы добавить позицию:
1.  На странице мониторинга очереди, точки обслуживания или планировщика очереди нажмите «<Add/>».
    :step_result: откроется страница добавления позиции.
    <ImageContainer src={"/images/working_with_checkpoint/add_position_step_mode.png"} description={"Страница добавления позиции, первый этап — выбор режима"}/>
2.  Выберите режим — «Добавить в очередь».
    :::info
    Данный режим предназначен для добавления клиента в очередь. Если нужно добавить технический перерыв сотруднику, см. раздел [Включение технического перерыва через добавление позиции](pause_maintenance/pause_maintenance_via_adding.mdx).
    :::
    :step_result: откроется блок заполнения контактной информации о клиенте.
    <ImageContainer src={"/images/working_with_checkpoint/add_position_step_contact_info.png"} description={"Страница добавления позиции, второй этап — контактная информация и услуги"}/>
3.  Заполните персональные данные о клиенте:
    1.  «Название юридического лица».
    2.  «Фамилия».
    3.  «Имя и отчество».
    4.  «Адрес электронной почты».
    5.  «Номер телефона».
    :::info
    Наличие полей персональных данных клиента зависит от настроек очереди, подробнее см. раздел [Настройка отображения персональных данных клиентов](../lines_management/line_settings/display_personal_data.mdx).
    :::
4.  Выберите язык, если в настройках места задано более одного языка. Возможность выбора языка задается в параметрах места, подробнее см. шаг [4.k](../shops_management/shops_creation_and_settings.mdx#substep_k) в разделе [Создание, настройка и удаление места](../shops_management/shops_creation_and_settings.mdx).
5.  Выберите услуги, если они включены в настройках очереди. При добавлении нескольких услуг суммарное время обслуживания складывается.
    :::info
    Несколько услуг можно выбрать только если включена функция в настройках очереди, подробнее см. раздел [Настройка, добавление и удаление услуг в очереди](../lines_management/line_settings/service_settings.mdx).
    :::
6.  Укажите количество человек. В зависимости от количества человек меняется время обслуживания. Максимальное количество человек может быть настроено для всех услуг и для каждой отдельной услуги.
7.  Заполните дополнительные поля, если они включены в настройках очереди. Дополнительные поля отображаются на
    сером фоне. Поддерживается сканирование дополнительных полей с QR-кода при помощи кнопки «<QrCodeScanner/>».
    <ImageContainer src={"/images/working_with_checkpoint/additional_fields.png"} description={"Пример дополнительных полей при добавлении позиции"}/>
8.  Чтобы позиция была приоритетнее и отображалась в очереди выше других позиций:
    1.  Раскройте блок «Дополнительные параметры».
    2.  Включите опцию «Приоритетное обслуживание».
    3.  Укажите комментарий персонала.
9.  Если необходимо указать конкретное время обслуживания и/или точку обслуживания:
    1.  Раскройте блок «Дополнительные параметры».
    2.  Задайте новое время в поле «Указанное время обслуживания».
    3.  Выберите точку обслуживания. Позиция будет назначена только к указанной точке обслуживания.
10. Нажмите «Далее».
    :step_result: откроется блок выбора времени начала обслуживания.
    <ImageContainer src={"/images/working_with_checkpoint/add_position_step_time.png"} description={"Страница добавления позиции, третий этап — временные интервалы"}/>
11. Выберите один из предложенных временных интервалов или включите режим «Как можно скорее», при помощи которого позиция добавится на самое ранее возможное время.
    :::tip
    Если доступного временного интервала нет, это значит, что очередь на данный момент заполнена или не имеет
    свободных слотов для позиции с такими настройками. Убедитесь, что настройки позиции верные.
    :::
    :::info
    Режим «Как можно скорее» можно выбрать только если он включен в настройках очереди, подробнее см. в разделе [Настройка основных параметров очереди](../lines_management/line_settings/main_settings.mdx).
    :::
12. Нажмите «Добавить».
    :step_result: откроется страница мониторинга, на странице появится добавленная позиция.
