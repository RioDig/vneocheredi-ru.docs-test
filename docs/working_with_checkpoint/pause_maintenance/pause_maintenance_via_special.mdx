# Включение технического перерыва через действие «Приостановить обслуживание» {#pause_maintenance_via_special}

:::caution
Включение технического перерыва возможно только если в настройках очереди в списке услуг есть технические услуги,
подробнее о настройке услуг см. в разделе [Настройка, добавление и удаление услуг в очереди](../../lines_management/line_settings/service_settings.mdx).
:::

Чтобы включить технический перерыв через действие «Приостановить обслуживание»:

1.  На главной странице или на странице «Места» выберите место и перейдите на страницу «Очереди» → выбранная очередь →
    «Точки обслуживания» → нажмите на точку обслуживания или на кнопку «<Visibility/>» для перехода к мониторингу нужной точки обслуживания.
    :step_result: откроется страница мониторинга точки обслуживания.
    <ImageContainer src={"/images/working_with_checkpoint/monitoring_page.png"} description={"Страница мониторинга точки обслуживания"}/>
2.  Нажмите «<MoreVert/>» в верхней части страницы.
    :step_result: откроется всплывающее окно с дополнительными действиями мониторинга.
    <ImageContainer src={"/images/working_with_checkpoint/monitoring_page_additional_actions.png"} description={"Дополнительные действия на странице мониторинга точки обслуживания"}/>
3.  Нажмите «Приостановить обслуживание».
    :step_result: откроется страница остановки обслуживания.
    <ImageContainer src={"/images/working_with_checkpoint/temporary_stop.png"} description={"Страница временной остановки точки обслуживания"}/>
4.  Выберите одну из технических услуг.
5.  Нажмите «Остановить обслуживание».
    :step_result: технический перерыв появится автоматически как позиция в статусе «Обслуживается», назначенная на пользователя, который выполнил приостановку обслуживания.
    <ImageContainer src={"/images/working_with_checkpoint/tech_position_in_progress.png"} description={"Позиция технического перерыва в статусе «Обслуживается»"}/>
6.  По истечении технического перерыва нажмите на карточку позиции и выполните действие «Закончить обслуживание».
    :step_result: внизу страницы появится всплывающее уведомление «Действие успешно выполнено», карточка позиции пропадет из списка позиций мониторинга точки обслуживания.
