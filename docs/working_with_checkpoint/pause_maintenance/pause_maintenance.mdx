# Включение технического перерыва {#pause_maintenance}

:::caution
Включение технического перерыва возможно только если в настройках очереди в списке услуг есть технические услуги,
подробнее о настройке услуг см. в разделе [Настройка, добавление и удаление услуг в очереди](../../lines_management/line_settings/service_settings.mdx).
:::

Технический перерыв — это временная остановка точки обслуживания, которая отображается как позиция с технической услугой в мониторинге.
