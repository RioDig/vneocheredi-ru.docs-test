# Добавление маячков места {#shops_beacons}

Маячки — это Bluetooth-устройства, которые позволяют контролировать местоположение клиентов места, если они используют мобильное приложение.

После физической установки маячка необходимо его добавить в Систему к нужному месту.

:::caution
Чтобы настраивать этот модуль, включите его при создании или редактировании места. Подробнее см. шаг [5](shops_creation_and_settings.mdx#step5) в разделе [Создание, настройка и удаление места](shops_creation_and_settings.mdx).
:::

Чтобы добавить маячок:
1.  На странице места перейдите на страницу «Маячки».
    <ImageContainer src={"/images/shops_management/beacons_page.png"} description={"Страница «Маячки»"}/>
2.  Нажмите «<Add/>».
    <ImageContainer src={"/images/shops_management/beacon_creation.png"} description={"Страница добавления маячка"}/>
3.  Заполните следующие поля:
    1.  «Название» — будет отображаться в списке маячков.
    2.  «Major» — Major номер маячка.
    3.  «Minor» — Minor номер маячка.
    4.  «Дата обслуживания» — дата последнего обслуживания маячка.
4.  Нажмите «Сохранить».
    :step_result: на странице «Маячки» появится добавленный маячок. При необходимости его можно выключить для обработки Системой.
