# Решение проблемы со звуком и автозапуском баннера типа «Видео» {#banner_autoplay_sound_fix}

Иногда на странице терминала «Экран вызова» рекламный баннер типа «Видео» с источником в виде ссылки, например на YouTube, может не запуститься автоматически или у него не будет звука. Для решения этой проблемы нужно настроить разрешения для сайта в браузере на устройстве, на котором открыта страница терминала.

Чтобы настроить разрешения сайта \(на примере браузера Google Chrome\):
1.  Перейдите на страницу экрана вызова.
    <ImageContainer src={"/images/troubleshooting/banner_audio_autoplay_fix/terminal_page.png"} description={"Страница экрана вызова"}/>
2.  Нажмите на кнопку информации о сайте. В Google Chrome — «![](/images/troubleshooting/banner_audio_autoplay_fix/site_info_icon.png)».
    :step_result: откроется всплывающее окно с информацией о сайте.
    <ImageContainer src={"/images/troubleshooting/banner_audio_autoplay_fix/site_info.png"} description={"Информация о сайте и ссылки для управления настройками сайта"}/>
3.  Нажмите «Настройки сайта».
    :step_result: в другой вкладке браузера откроется страница управления разрешениями сайта.
    <ImageContainer src={"/images/troubleshooting/banner_audio_autoplay_fix/site_permissions_page.png"} description={"Страница настройки разрешений сайта"}/>
4.  Найдите разрешение «Звук» и выберите опцию «Разрешить».
5.  Вернитесь на страницу экрана вызова и перезагрузите страницу, чтобы изменения вступили в силу.
    :step_result: теперь все рекламные баннеры типа «Видео» будут автоматически воспроизводиться со звуком.
