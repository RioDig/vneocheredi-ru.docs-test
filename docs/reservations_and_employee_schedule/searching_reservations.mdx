# Поиск предварительных записей {#searching_reservations}

Система позволяет искать предварительные записи для выполнения различных действий.

Чтобы найти предварительную запись:
1.  На главной странице откройте выбранное место и перейдите на страницу «Очереди» → выбранная очередь → «Бронирования».
    :step_result: откроется страница «Бронирования».
    <ImageContainer src={"/images/reservations_and_employee_schedule/reservations_page.png"} description={"Страница «Бронирования»"}/>
2.  Укажите условия поиска:
    1.  Текстовое поле «Поиск» для поиска по коду бронирования, персональным данным клиента, идентификатору записи и т.д.
    2.  Поиск по дате в полях «Дата от» и «Дата до».
3.  При необходимости можно скрыть удаленные предварительные записи.
4.  Нажмите «Скачать бронирования», чтобы скачать файл формата XLSX со списком всех предварительных записей, подходящих под условия поиска.
5.  Нажмите «Поиск».
    :step_result: по заданным критериям поиска будут найдены предварительные записи.
