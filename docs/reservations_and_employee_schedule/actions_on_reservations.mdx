# Действия с предварительными записями {#actions_on_reservations}

<ImageContainer src={"/images/reservations_and_employee_schedule/reservations_page_actions.png"} description={"Предварительная запись на странице «Бронирования»"}/>

У предварительной записи могут быть следующие действия:
1.  «Редактировать» — изменение деталей предварительной записи: услуги, количество \(пакетов документов, человек, услуг, объектов\), внутренний комментарий, время предварительной записи.
    :::caution
    Для избежания возможности передачи предварительной записи другому лицу редактировать персональные данные существующей предварительной записи невозможно.
    :::
2.  «Информационные тэги» — указать теги и комментарий к предварительной записи.
    :::info
    Возможность указывать теги доступна только если эта функция включена в настройках очереди, подробнее см. раздел [Настройка информационных тэгов](../lines_management/line_settings/additional_info.mdx).
    :::
3.  «Подтвердить» — перевести предварительную запись в статус «Подтверждена».
4.  «Удалить» — удаление предварительной записи.
5.  «Печать» — перейти к версии для печати.
6.  «Журнал» — отобразить запись в журнале по этой предварительной записи.
