# Добавление предварительной записи клиента {#adding_reservations}

Чтобы добавить предварительную запись:
1.  На главной странице откройте выбранное место и перейдите на страницу «Очереди» → выбранная очередь → «Бронирования».
    :step_result: откроется страница «Бронирования».
    <ImageContainer src={"/images/reservations_and_employee_schedule/reservations_page.png"} description={"Страница «Бронирования»"}/>
2.  Нажмите «Добавить бронирование»
    <ImageContainer src={"/images/reservations_and_employee_schedule/add_reservation_step_line_info.png"} description={"Добавление предварительной записи, первый этап — выбор очереди и услуги"}/>
3.  Выберите услугу и количество \(пакетов документов, человек, услуг, объектов\).
4.  При необходимости введите внутренний комментарий, который отобразится в деталях предварительной записи только пользователям Системы.
5.  Нажмите «Далее».
    <ImageContainer src={"/images/reservations_and_employee_schedule/add_reservation_step_time.png"} description={"Добавление предварительной записи, второй этап — выбор времени"}/>
6.  Выберите дату и время для предварительной записи. Возможность выбора даты и времени зависит от графика сотрудников, подробнее см. в разделе [Настройка графика сотрудников](setting_employee_schedule.mdx).
7.  Нажмите «Далее».
    <ImageContainer src={"/images/reservations_and_employee_schedule/add_reservation_step_client.png"} description={"Добавление предварительной записи, третий этап — заполнение данных клиента"}/>
8.  Заполните данные о клиенте одним из способов:
    1.  Начните вводить номер телефона и выберите клиента, который был в очереди ранее.
    2.  Заполните все поля для нового клиента.
9.  Нажмите «Далее».
    <ImageContainer src={"/images/reservations_and_employee_schedule/add_reservation_step_confirm.png"} description={"Добавление предварительной записи, четвертый этап — подтверждение предварительной записи"}/>
10. При необходимости распечатайте контактную информацию для проверки сейчас или включите «Распечатать после создания» для печати позже.
11. Нажмите «Создать».
    :step_result: будет создана позиция клиента на указанный день.
