# Поиск свободного времени для предварительной записи {#search_free_time}

Функция поиска свободного времени — это один из способов создать предварительную запись. Эта функция обычно используется сотрудниками колл-центров при запросе клиента записаться в очередь на ближайшую дату.

Чтобы создать предварительную запись через функцию поиска свободного времени:
1.  С главной страницы перейдите на страницу «Бронирования».
    :step_result: откроется страница просмотра предварительных записей и управления ими.
    <ImageContainer src={"/images/reservations_and_employee_schedule/reservations_page_from-main-page.png"} description={"Страница «Бронирования»"}/>
2.  Нажмите «Искать свободное время».
    :step_result: откроется страница поиска свободного времени для предварительной записи.
    <ImageContainer src={"/images/reservations_and_employee_schedule/reservations_search_time.png"} description={"Страница поиска свободного времени"}/>
3.  Выберите нужную услугу из списка и укажите количество \(участников, пакетов документов и др.\), затем нажмите «Найти».
    :step_result: при наличии свободного времени для предварительной записи на указанную услугу появится список временных интервалов очередей с ближайшими датами и временем.
    <ImageContainer src={"/images/reservations_and_employee_schedule/reservations_search_time_step2.png"} description={"Свободные интервалы предварительной записи для услуги «Получение готового результата»"}/>
4.  Выберите один из временных интервалов. При необходимости воспользуйтесь фильтрацией по очереди и строкой поиска, чтобы найти нужный временной интервал.
    :step_result: откроется страница создания предварительной записи с предустановленными полями: место, очередь, услуга, количество и время.
    <ImageContainer src={"/images/reservations_and_employee_schedule/reservations_search_time_step3.png"} description={"Страница создания предварительной записи"}/>
5.  Выберите время предварительной записи и нажмите «Далее».
    :::note
    При необходимости нажмите «Назад» и скорректируйте место, очередь, услугу и количество.
    :::
    :step_result: откроется вкладка для заполнения данных о клиенте.
    <ImageContainer src={"/images/reservations_and_employee_schedule/add_reservation_step_client.png"} description={"Создание предварительной записи — заполнение данных клиента"}/>
6.  Во вкладке «Клиент» заполните персональные данные клиента одним из способов:
    1.  Начните вводить номер телефона и выберите клиента, который был в очереди ранее.
    2.  Заполните все поля для нового клиента: номер телефона, ФИО, адрес электронной почты.
7.  Нажмите «Далее».
    :step_result: откроется вкладка подтверждения данных предварительной записи.
    <ImageContainer src={"/images/reservations_and_employee_schedule/add_reservation_step_confirm.png"} description={"Создание предварительной записи — подтверждение предварительной записи"}/>
8.  Убедитесь в правильности введенных данных о предварительной записи и клиенте. При необходимости распечатайте контактную информацию для проверки сейчас или включите «Распечатать после создания» для печати позже.
9.  Нажмите «Создать». Дождитесь появления уведомления внизу страницы, иначе предварительная запись не будет создана.
    :step_result: внизу страницы появится всплывающее уведомление «Бронирование успешно создано».
