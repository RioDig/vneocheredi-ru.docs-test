# Создание и удаление точки обслуживания {#checkpoint_creation}

import ImageContainer from "../../src/components/ImageContainer";
import {Add, RemoveCircleOutline, Edit} from "@mui/icons-material";

:::info
Создание и удаление точек обслуживания доступно только пользователям типа «Администратор» с
правом доступа «Доступ к разделу "Настройки места"», подробнее см. в разделе [Права доступа](../account_management/access_rights.mdx).
:::

В этом разделе:

-   [Удаление точки обслуживания](#delete_checkpoint)

## Создание точки обслуживания {#create_checkpoint}

Чтобы создать точку обслуживания:

1.  На главной странице или на странице «Места» выберите место и перейдите на страницу «Очереди» → выбранная очередь → «Точки обслуживания».
    :step_result: откроется страница со списком точек обслуживания.
    <ImageContainer src={"/images/checkpoint_management/checkpoint_page.png"} description={"Страница «Точки обслуживания»"}/>
2.  Нажмите «<Add/>» в верхней части страницы.
    :step_result: откроется страница создания точки обслуживания.
    <ImageContainer src={"/images/checkpoint_management/create_checkpoint.png"} description={"Страница создания точки обслуживания"}/>
3.  Заполните информацию о точке обслуживания в следующих полях:
    1.  «Название точки обслуживания» — отображается в интерфейсе административной панели, на терминалах и у клиентов после успешной записи.
    2.  «Описание».
4.  При необходимости включите опцию аудиозаписи разговоров во время обслуживания клиента.
    :::info
    Опция аудиозаписи разговоров доступна только если она включена в настройках очереди,
    подробнее см. шаг [3](../lines_management/line_settings/advanced_settings.mdx#step3)
    в разделе [Расширенные настройки очереди](../lines_management/line_settings/advanced_settings.mdx).
    :::
5.  Если у очереди задан алгоритм расчета — «Групповые бронирования», то при создании точки обслуживания можно задать проходимость:
    1.  Включите «Задать проходимость».
    2.  Раскройте блок «Проходимость».
    3.  Нажмите «Добавить».
        :step_result: появится строка для заполнения проходимости для одной даты.
        <ImageContainer src={"/images/checkpoint_management/passability_settings.png"} description={""}/>
    4.  Заполните данные о проходимости:
        1.  «Дата» — время, для которого указывается проходимость.
        2.  «Проходимость» — количество человек, которое предполагается в указанное время.
        3.  «Овербукинг» — количество человек, допускаемое поверх лимита проходимости.
        4.  «Запас».
        :::tip
        Удалить лишние даты можно нажатием на «<RemoveCircleOutline/>».
        :::
    :::info
    Задать алгоритм расчета очереди можно в настройках очереди, подробнее см. шаг [4](../lines_management/line_settings/main_settings.mdx#step4) в разделе [Настройка основных параметров очереди](../lines_management/line_settings/main_settings.mdx).
    :::
6.  Нажмите «Сохранить».
    :step_result: будет создана новая точка обслуживания и откроется страница «Точки обслуживания» в текущей очереди.


## Удаление точки обслуживания {#delete_checkpoint}

Чтобы удалить точку обслуживания:
1.  На главной странице или на странице «Места» выберите место и перейдите на страницу «Очереди» → выбранная очередь → «Точки обслуживания».
    :step_result: откроется страница со списком точек обслуживания.
    <ImageContainer src={"/images/checkpoint_management/checkpoint_page.png"} description={"Страница «Точки обслуживания»"}/>
2.  Нажмите «<Edit/>», чтобы перейти к редактированию точки обслуживания.
    :step_result: откроется страница редактирования точки обслуживания.
    <ImageContainer src={"/images/checkpoint_management/checkpoint_edit_page.png"} description={"Редактирование точки обслуживания"}/>
3.  Нажмите «Удалить» и подтвердите действие.
    :step_result: в нижней части страницы появится всплывающее уведомление «Точка обслуживания успешно удалена» и откроется страница «Точки обслуживания» в текущей очереди.
