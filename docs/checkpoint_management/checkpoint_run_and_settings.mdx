# Настройка и запуск точки обслуживания {#checkpoint_run_and_settings}

import ImageContainer from "../../src/components/ImageContainer";
import {PlayArrow} from "@mui/icons-material";


:::info
Запуск точек обслуживания доступен только пользователям с правом доступа «Открывать точки обслуживания», подробнее см. в разделе [Права доступа](../account_management/access_rights.mdx).
:::

Перед началом работы точки обслуживания ее необходимо настроить и запустить.

Чтобы настроить и запустить точку обслуживания:

1.  На главной странице или на странице «Места» выберите место и перейдите на страницу «Очереди» → выбранная очередь → «Точки обслуживания».
    :step_result: откроется страница со списком точек обслуживания.
    <ImageContainer src={"/images/checkpoint_management/checkpoint_page.png"} description={"Страница «Точки обслуживания»"} />
2.  Выберите нужную точку обслуживания и нажмите «<PlayArrow/>».
    :step_result: откроется страница параметров точки обслуживания.
    <ImageContainer src={"/images/checkpoint_management/checkpoint_running.png"} description={"Страница параметров только что созданной или остановленной точки обслуживания"}/>
3.  Введите имя и отчество сотрудника, который будет обслуживать эту точку обслуживания.
    :::note
    Имена сотрудников сохраняются в системе и будут предлагаться при вводе имени.
    :::
4.  Выберите один из четырех режимов запуска точки обслуживания:
    1.  «Запуск сейчас» — точка обслуживания будет запущена сразу и остановлена в указанное пользователем время.
        <ImageContainer src={"/images/checkpoint_management/checkpoint_running_mode_now.png"} description={"Страница запуска точки обслуживания, режим «Запустить сейчас»"}/>
    2.  «Запустить в установленные часы» — точка обслуживания будет запущена и остановлена в указанное пользователем время.
        <ImageContainer src={"/images/checkpoint_management/checkpoint_running_mode_planned.png"} description={"Страница запуска точки обслуживания, режим «Запустить в установленные часы»"}/>
    3.  «Запустить в часы работы очереди» — точка обслуживания будет работать по часам работы очереди, в которой находится эта точка обслуживания.
        :::info
        Часы работы очереди задаются в настройках очереди, подробнее см. в разделе [Настройка времени открытия очереди](../lines_management/line_settings/schedule_settings.mdx).
        :::
        <ImageContainer src={"/images/checkpoint_management/checkpoint_running_mode_line_scheduled.png"} description={"Страница запуска точки обслуживания, режим «Запустить в часы работы очереди»"}/>
    4.  «Запустить круглосуточно» — точка обслуживания будет работать круглосуточно.
        :::info
        Этот режим доступен только если в настройках очереди включена опция «Круглосуточно», подробнее см. в разделе [Настройка времени открытия очереди](../lines_management/line_settings/schedule_settings.mdx).
        :::
        <ImageContainer src={"/images/checkpoint_management/checkpoint_running_mode_everyday.png"} description={"Страница запуска точки обслуживания, режим «Круглосуточно»"}/>
5.  Заполните следующие поля:
    1.  «Время открытия» — указывается только при выборе режима «Запустить в установленные часы».
    2.  «Время закрытия» — указывается при выборе режимов «Запуск сейчас» и «Запустить в установленные часы».
    3.  «Перерыв» — при необходимости включите перерыв и укажите временной интервал. В это время точка обслуживания будет недоступна для обслуживания клиентов.
6.  Выберите услуги, которые будет оказывать эта точка обслуживания.
    <ImageContainer src={"/images/checkpoint_management/checkpoint_running_services.png"} description={"Выбор услуг при запуске точки обслуживания"}/>
    :::info
    При включении опции «Все услуги» точка обслуживания сможет обрабатывать все услуги, включая те, которые будут добавлены в очередь в будущем.
    :::
    :::caution
    Выбор услуг для точки обслуживания будет недоступен, если в настройках очереди нет услуг или параметр «Тип отображения услуг» установлен «Нет», подробнее о настройке услуг см. в разделе [Настройка, добавление и удаление услуг в очереди](../lines_management/line_settings/service_settings.mdx).
    :::
7.  Нажмите «Запустить».
    :step_result: внизу страницы появится всплывающее уведомление «Настройки успешно сохранены» и откроется страница «Точки обслуживания».
