## Редактирование организации {#edit}

Чтобы изменить название организации или настройки ограничений:
1.  С главной страницы ВнеОчереди перейдите на страницу «Организации».
    :step_result: откроется страница со списком организаций.
    <ImageContainer src={"/images/organizations_management/org_page.png"} description={"Страница «Организации»"}/>
2.  Выберите из списка нужную организацию и нажмите на нее.
    :step_result: откроется страница редактирования организации.
    <ImageContainer src={"/images/organizations_management/org_page_edit.png"} description={"Страница редактирования организации"}/>
3.  При необходимости измените название организации во вкладке «Параметры».
4.  При необходимости измените ограничения организации. Подробнее об ограничениях см. в разделе [Виды ограничений записи в организации](org_limitations.mdx).
5.  Нажмите «Сохранить», чтобы применить изменения.
    :step_result: внизу страницы появится всплывающее уведомление «Организация успешно обновлена».


## Удаление организации {#delete}

Чтобы удалить неиспользуемую организацию:
1.  С главной страницы ВнеОчереди перейдите на страницу «Организации».
    :step_result: откроется страница со списком организаций.
    <ImageContainer src={"/images/organizations_management/org_page.png"} description={"Страница «Организации»"}/>
2.  Выберите из списка нужную организацию и нажмите на нее.
    :step_result: откроется страница редактирования организации.
    <ImageContainer src={"/images/organizations_management/org_page_edit.png"} description={"Страница редактирования организации"}/>
3.  Перейдите на вкладку «Прочее», нажмите «Удалить» и подтвердите действие.
    :step_result: внизу страницы появится всплывающее уведомление «Организация успешно удалена».
