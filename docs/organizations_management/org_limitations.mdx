# Виды ограничений записи в организации {#org_limitations}

В этом разделе приведено описание ограничений на запись, которые указываются для организации при ее создании или редактировании.

<ImageContainer src={"/images/organizations_management/org_limitations.png"} description={"Настройки ограничений организации"}/>

<table>
    <thead>
    <tr>
        <th>Название ограничения</th>
        <th>Описание</th>
        <th>Настройки ограничения</th>
    </tr>
    </thead>
    <tbody>
    <tr>
        <td>Ограничить количество/время записей во все офисы</td>
        <td>Параметр задает ограничения для записей которые может одновременно иметь один клиент на все офисы организации</td>
        <td>
            -   Максимальное количество часов — <strong>число</strong> или <strong>не ограничено (пустое поле)</strong>. Максимальная сумма времени обслуживания всех записей одновременно у одного клиента во всех местах организации
            -   Максимальное количество записей — <strong>число (минимально 1)</strong>. Максимальное количество записей одновременно у одного клиента во всех местах организации
        </td>
    </tr>
    <tr>
        <td>Ограничить количество офисов</td>
        <td>Параметр ограничивает количество офисов в которые клиент может быть записан одновременно</td>
        <td>
            -   В один день — <strong>число</strong> или <strong>не ограничено (пустое поле)</strong>. Максимальное количество мест, в которые одновременно записан один человек в один день
            -   Всего — <strong>число</strong> или <strong>не ограничено (пустое поле)</strong>. Максимальное количество мест, в которые одновременно записан один человек всего
        </td>
    </tr>
    <tr>
        <td>Ограничить количество записей в один офис</td>
        <td>Параметр ограничивает количество записей которые клиент может иметь одновременно в одном офисе</td>
        <td>
            -   В один день — <strong>число</strong> или <strong>не ограничено (пустое поле)</strong>. Максимальное одновременное количество записей одного человека в один день в одном месте
            -   Всего — <strong>число</strong> или <strong>не ограничено (пустое поле)</strong>. Максимальное одновременное количество записей одного человека всего в одном месте
        </td>
    </tr>
    <tr>
        <td>Ограничить запись с одного устройства разных клиентов</td>
        <td>Параметр ограничивает количество клиентов которые могут записаться с одного устройства: браузера или мобильного телефона</td>
        <td>
            -   Можно изменить — <strong>ничего/только ФИО/все контактные данные</strong>. Ограничивает возможность редактирования записи
            -   Максимальное количество разных клиентов — <strong>число</strong> или <strong>не ограничено (пустое поле)</strong>. Количество записей с разными персональными данными, сделанных с одного устройства
        </td>
    </tr>
    <tr>
        <td>Ограничить редактирование предварительных записей</td>
        <td>Параметр ограничивает возможности редактирования предварительных записей</td>
        <td>
            -   Максимальное количество редактирований — <strong>число</strong> или <strong>не ограничено (пустое поле)</strong>. Число раз сколько можно выполнять редактирование
            -   Возможность изменения офиса — <strong>да/нет/пока не настал день записи</strong>. Дает возможность изменить место
            -   Возможность изменения услуги — <strong>да/нет/пока не настал день записи</strong>.
            -   Возможность изменения даты и времени — <strong>да/нет/пока не настал день записи</strong>. Дает возможность изменить временной слот или дату оказания услуги.
        </td>
    </tr>
    <tr>
        <td>Ограничить редактирование записей день в день</td>
        <td>Параметр ограничивает возможности редактирования записей созданных день в день (на сегодня)</td>
        <td>
            -   Максимальное количество редактирований — <strong>число</strong> или <strong>не ограничено (пустое поле)</strong>. Число раз сколько можно выполнять редактирование
            -   Возможность изменения офиса — <strong>да/нет</strong>. Дает возможность изменить место
            -   Возможность изменения услуги — <strong>да/нет</strong>.
            -   Возможность изменения даты и времени — <strong>да/нет</strong>. Дает возможность изменить временной слот или дату оказания услуги.
        </td>
    </tr>
    <tr>
        <td>Запретить несколько записей в одно и то же время</td>
        <td>Параметр ограничивает для клиента возможность записаться на то время, на которое он уже записан и планируется быть в обслуживании</td>
        <td>Нет дополнительных настроек</td>
    </tr>
    </tbody>
</table>
