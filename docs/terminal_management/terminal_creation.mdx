# Создание и подключение терминала {#terminal_creation}

В этом разделе приведены инструкции по созданию и подключению к месту терминалов разного типа.

## Экран вызова {#infopad}

Чтобы создать и подключить экран вызова:
1.  На главной странице перейдите на страницу «Терминалы».
2.  Нажмите «<Add/>».
3.  Выберите тип терминала — «Экран вызова».
    <ImageContainer src={"/images/terminal_management/infopad_creation.png"} description={"Страница создания терминала типа «Экран вызова»"}/>
4.  Выберите один из режимов вызова:
    1.  «Только анимация \(без звука\)» — звуковой или голосовой вызов не осуществляется.
    2.  «Звуковой сигнал» — только звуковой вызов позиции.
    3.  «Смешанный режим» — режим, при котором сначала производится звуковой вызов, а через указанное количество раз начинается голосовой вызов.
        :::info
        Количество раз, с которого начинается голосовой вызов, указывается в поле «С какого раза начать голосовой вызов».
        :::
    4.  «Голосовой вызов» — только голосовой вызов позиции.
5.  Заполните следующие поля:
    1.  «Название» — будет отображаться в списке терминалов.
    2.  «Краткое название».
    3.  «Разрешение экрана по высоте» и «Разрешение экрана по ширине» — используются для определения размера экрана вызова.
    4.  «С какого раза начать голосовой вызов» — количество звуковых сигналов, после которых начнутся голосовые вызовы.
        :::note
        Доступно только при включенном режиме «Смешанный режим».
        :::
    5.  «Количество повторений после первого вызова» — количество звуковых или голосовых вызовов после первого.
    6.  «Интервал между повторениями вызовов».
    7.  «Количество отображений позиций в ожидании» — максимальное количество позиций в ожидании вызова на точку обслуживания.
    8.  «Максимальное время ожидания для показа» — время \(в секундах\), после которого позиция исчезнет с экрана вызова.
6.  При необходимости включите отображение информации о позиции:
    1.  «Показывать имя и первую букву фамилии».
    2.  «Показывать фамилию».
    3.  «Показывать код брони».
7.  Включите опцию «Отображать номер талона», чтобы на экране вызова у позиций, которые были созданы через терминал, показывался номер талона.
8.  При необходимости отключите использование соединения SignalR.
9.  Включите опцию «Отключить визуальные эффекты для улучшения производительности», чтобы включить экран вызова в упрощенном режиме без анимаций.
10. При необходимости отключите отображение логотипа в правом нижнем углу экрана.
11. Настройте отображение индивидуального логотипа:
    1.  Включите опцию «Отображать индивидуальный логотип», чтобы изменить стандартный логотип ВнеОчереди.
    2.  Загрузите файл в поле «Логотип».
        :::note
        -   Поддерживаемые форматы: SVG, PNG, JPG.
        -   Ограничение веса файла: 10 Мб.
        -   Ограничение разрешения файла: нет.
        -   Максимальное соотношение сторон: 16:9 \(при превышении файл автоматически масштабируется в 16:9\)
        :::
12. Укажите в поле «Увеличение» коэффициент увеличения текста на экране вызова. По умолчанию — 1.
13. Включите «Объединение очередей» и укажите заголовок, если нужно объединить все позиции из двух и более очередей в один список.
14. Выберите места и очереди, в которых нужно отслеживать позиции для экрана вызова.
15. Нажмите «Сохранить».
    :step_result: откроется страница «Терминалы» с добавленным терминалом.
16. Включите терминал для начала его работы.
    :::tip
    Получить ссылку на терминал типа «Экран вызова» можно в блоке «Ссылки» на странице терминала.
    :::


## Веб-терминал {#kiosk}

Чтобы создать и подключить терминал типа «Веб-терминал»:
1.  На главной странице перейдите на страницу «Терминалы».
2.  Нажмите «<Add/>».
3.  Выберите тип терминала — «Веб-терминал».
    <ImageContainer src={"/images/terminal_management/kiosk_creation.png"} description={"Страница создания терминала типа «Веб-терминал»"}/>
4.  Заполните следующие поля:
    1.  «Название» — будет отображаться в списке терминалов.
    2.  «Краткое название».
    3.  «Отображаемое название» — название, которое будет отображаться на странице терминала.
    4.  «Разрешение экрана по высоте» и «Разрешение экрана по ширине» — используются для определения размера экрана терминала.
    5.  «Начинать бронирование с» — время, с которого доступна запись через терминал.
5.  Чтобы включить экран с информацией после записи через терминал:
    1.  Включите опцию «Показывать финальный экран с информацией о записи».
        :step_result: появится опция «Автоматически скрывать после показа».
    2.  Чтобы финальный экран закрывался автоматически спустя время, включите опцию «Автоматически скрывать после показа» и укажите время показа финального экрана в секундах в появившемся ниже поле.
    :::info
    Для изменения текста на финальном экране отредактируйте шаблон сообщения «Поздравления с присоединением к очереди через терминал», подробнее см. в разделе [Изменение шаблонов сообщений](../message_templates/message_templates_setting.mdx).
    :::
6.  При необходимости включите возможность печати бумажных билетов:
    1.  В блоке настроек «Оборудование» включите опцию «Термопринтер».
    2.  Укажите максимальную вместимость билетов.
    3.  Укажите количество билетов, после которого будет уведомление.
7.  Выберите места и очереди, в которых нужно отслеживать позиции для экрана вызова.
8.  Нажмите «Сохранить».
    :step_result: откроется страница «Терминалы» с добавленным терминалом.
9.  Включите терминал для начала его работы.
    :::tip
    Ссылка на терминал типа «Веб-терминал» выглядит следующим образом:<br/>
    `https://<ваш домен>/terminal/?terminalId={GUID}&languages=ru`<br/>
    Чтобы добавить клиенту возможность проверить свою запись и отметиться в офисе \(действие «На месте»\), используйте следующую ссылку:<br/>
    `https://<ваш домен>/terminal/?terminalId={GUID}&languages=ru&mfcActions=true`<br/>
    **\{GUID\}** — это идентификатор терминала, который можно найти на странице терминала.
    Подробнее о других параметрах ссылки см. в разделе [Ссылка на терминал типа «Веб-терминал»](../instructions/link_formation.mdx#).
    :::
