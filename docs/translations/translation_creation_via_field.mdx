# Добавление перевода при помощи полей {#translation_creation_via_field}

:::info
Чтобы просматривать список переводов, изменять, создавать или удалять их, пользователь должен обладать правом «Доступ к разделу "Переводы"». Подробнее о правах пользователей см. в разделе [Права доступа](../account_management/access_rights.mdx).
:::

Для быстрого создания и редактирования перевода можно использовать кнопку «<Public/>» рядом с полем в административной панели ВнеОчереди.

Чтобы добавить перевод через кнопку рядом с полем, поддерживающим вставку переводов:
1.  Заполните поле, для которого нужно добавить перевод.
    <ImageContainer src={"/images/translations/field_with_translation.png"} description={"Поле с возможностью добавления перевода"}/>
    :::tip
    Для удобного дальнейшего поиска перевода в списке на странице «Переводы» в поле можно ввести исходный текст перевода, а затем настроить переводы на разные языки.
    :::
2.  Нажмите «<Public/>».
    :step_result: появится окно добавления перевода.
    <ImageContainer src={"/images/translations/translation_popup.png"} description={"Окно добавления перевода"}/>
3.  Задайте переводы на разные языки:
    1.  Нажмите «Добавить перевод».
        :step_result: появится поле с выбором языка и текстом перевода.
        <ImageContainer src={"/images/translations/translation_popup_added_language.png"} description={"Добавление перевода на другой язык"}/>
    2.  Из выпадающего списка выберите язык, на который нужно перевести исходный текст.
    3.  Введите текст перевода.
    4.  Добавьте переводы на другие языки аналогичным образом. Чтобы удалить лишние переводы, нажмите «<RemoveCircleOutline/>».
4.  Нажмите «Сохранить».
    :step_result: перевод будет создан и появится на странице «Переводы».
