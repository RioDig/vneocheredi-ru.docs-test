# Редактирование и удаление перевода {#translation_edit_delete}

:::info
Чтобы просматривать список переводов, изменять, создавать или удалять их, пользователь должен обладать правом «Доступ к разделу "Переводы"». Подробнее о правах пользователей см. в разделе [Права доступа](../account_management/access_rights.mdx).
:::

В этом разделе:
-   [Редактирование перевода](#edit)
-   [Удаление перевода](#delete)


## Редактирование перевода {#edit}

Чтобы изменить текст перевода или добавить ему новый язык:
1.  С главной страницы перейдите на страницу «Переводы».
    :step_result: откроется страница со списком переводов.
    <ImageContainer src={"/images/translations/translations_page.png"} description={"Страница «Переводы»"}/>
2.  Выберите из списка нужный перевод и нажмите на него.
    :step_result: откроется страница редактирования перевода.
    <ImageContainer src={"/images/translations/translation_edit.png"} description={"Страница редактирования перевода"}/>
3.  Отредактируйте перевод:
    1.  Измените поле «Исходный текст» — текст, который будет заменяться переводом в зависимости от языка.
        :::info
        При изменении исходного текста перевода он перестанет работать в тех местах, где использовался исходный текст.
        :::
        :::tip
        В качестве исходного текста лучше использовать не переведенную фразу, а переменную, на место которой будет вставлен перевод. Например, исходный текст «\[UGMU\]», вместо которого будут вставляться переводы:
        -   На русский — Уральский государственный медицинский университет
        -   На английский — Ural State Medical University
        :::
    2.  Измените переведенный текст на разные языки. При необходимости добавьте перевод на новый язык, нажав на кнопку «Добавить перевод». Чтобы удалить лишние переводы, нажмите «<RemoveCircleOutline/>».
4.  Нажмите «Сохранить», чтобы применить изменения.
    :step_result: внизу страницы появится всплывающее уведомление «Переводы успешно обновлены».


## Удаление перевода {#delete}

Чтобы удалить неиспользуемый перевод:
1.  С главной страницы перейдите на страницу «Переводы».
    :step_result: откроется страница со списком переводов.
    <ImageContainer src={"/images/translations/translations_page.png"} description={"Страница «Переводы»"}/>
2.  Выберите из списка нужный перевод и нажмите на него.
    :step_result: откроется страница редактирования перевода.
    <ImageContainer src={"/images/translations/translation_edit.png"} description={"Страница редактирования перевода"}/>
3.  Нажмите «Удалить» и подтвердите действие.
    :step_result: внизу страницы появится всплывающее уведомление «Переводы успешно обновлены».
