# Добавление перевода на странице «Переводы» {#translation_creation_via_page}

На странице «Переводы» можно создавать переводы для определенной организации и просматривать уже созданные, чтобы отредактировать или удалить перевод.

:::info
Чтобы просматривать список переводов, изменять, создавать или удалять их, пользователь должен обладать правом «Доступ к разделу "Переводы"». Подробнее о правах пользователей см. в разделе [Права доступа](../account_management/access_rights.mdx).
:::

Чтобы добавить перевод через страницу «Переводы»:
1.  С главной страницы перейдите на страницу «Переводы».
    :step_result: откроется страница со списком переводов.
    <ImageContainer src={"/images/translations/translations_page.png"} description={"Страница «Переводы»"}/>
2.  Нажмите «<Add/>».
    :step_result: откроется страница создания перевода.
    <ImageContainer src={"/images/translations/translation_create_page.png"} description={"Страница создания перевода"}/>
3.  В поле «Исходный текст» введите текст, который будет заменяться переводом в зависимости от языка.
    :::tip
    В качестве исходного текста лучше использовать не переведенную фразу, а переменную, на место которой будет вставлен перевод. Например, исходный текст «\[UGMU\]», вместо которого будут вставляться переводы:
    -   На русский — Уральский государственный медицинский университет
    -   На английский — Ural State Medical University
    :::
4.  Выберите организацию из выпадающего списка. Перевод будет использоваться только для мест выбранной организации. Подробнее об организациях см. в разделе [Управление организациями](../organizations_management/organizations_management.mdx).
5.  Добавьте переводы:
    1.  Нажмите «Добавить перевод».
        :step_result:появится поле с выбором языка и текстом перевода.
        <ImageContainer src={"/images/translations/translation_create_page_added_language.png"} description={"Добавлен язык для перевода"}/>
    2.  Из выпадающего списка выберите язык, на который нужно перевести исходный текст.
    3.  Введите текст перевода.
    4.  Добавьте переводы на другие языки аналогичным образом. Чтобы удалить лишние переводы, нажмите «<RemoveCircleOutline/>».
6.  Нажмите «Сохранить».
    :step_result: откроется страница «Переводы» с добавленным переводом.
    :::tip
    Для использования перевода введите исходный текст перевода в поле, где есть такая возможность.
    :::
