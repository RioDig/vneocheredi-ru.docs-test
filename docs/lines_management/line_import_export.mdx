# Импорт/экспорт настроек существующей очереди {#line_import_export}

Для быстрого редактирования множества очередей по одному шаблону можно воспользоваться функцией импорта и экспорта настроек очереди. Настройки очереди можно экспортировать в файл, чтобы импортировать в другую существующую очередь для изменения настроек. Также файл настроек можно использовать при создании очереди, подробнее см. в разделе [Создание очереди с импортированными настройками](line_creation/line_creation_with_import.mdx).

В этом разделе:
-   [Экспорт настроек очереди](#export)
-   [Импорт настроек в существующую очередь](#import)

## Экспорт настроек очереди {#export}

Перед импортом настроек в существующую очередь нужно получить файл формата JSON. Для этого:
1.  На главной странице или на странице «Места» выберите место и перейдите на страницу «Очереди» → выбранная очередь → «Параметры» → «Редактировать» → группа настроек «Расширенные настройки».
    <ImageContainer src={"/images/lines_management/line_settings/advanced_settings.png"} description={"Группа «Расширенные настройки» в настройках очереди"}/>
2.  Нажмите «Экспортировать» в поле «Экспортировать как шаблон».
    :step_result: начнется скачивание файла формата JSON со списком настроек очереди.


## Импорт настроек в существующую очередь {#import}

Для импорта настроек в существующую очередь нужен файл формата JSON с настройками другой очереди, подробнее о его создании см. в разделе [Экспорт настроек очереди](#export).

Чтобы импортировать настройки в очередь:
1.  На главной странице или на странице «Места» выберите место и перейдите на страницу «Очереди» → выбранная очередь → «Параметры» → «Редактировать» → группа настроек «Расширенные настройки».
    <ImageContainer src={"/images/lines_management/line_settings/advanced_settings.png"} description={"Группа «Расширенные настройки» в настройках очереди"}/>
2.  Нажмите «Импортировать» в поле «Импортировать из шаблона».
    :step_result: откроется окно с предложением выбрать файл.
3.  Выберите файл формата JSON, полученный после экспорта настроек другой очереди.
    :step_result: появится окно импорта настроек очереди.
    <ImageContainer src={"/images/lines_management/line_export_import/import_settings_page.png"} description={"Окно импорта настроек"}/>
4.  Выберите все параметры из шаблона настроек, которые вы хотите применить к очереди. Обратите внимание, что выбрать можно только различающиеся параметры. Для удобства включите опцию «Только различающиеся» в левом верхнем углу. Количество настроек, которые будут применены к очереди, отображается в правом нижнем углу.
5.  Нажмите «Применить» в правом нижнем углу окна импорта настроек очереди.
    :step_result: выбранные настройки применятся к очереди.
6.  Нажмите «Сохранить», чтобы применить изменения.
    :step_result: внизу страницы появится всплывающее уведомление «Очередь успешно обновлена».
