# Настройка дополнительных полей очереди {#additional_fields}

Системой предусмотрено создание дополнительных полей, которые заполняются клиентом при записи в очередь или сотрудником при добавлении позиции. Поля могут использоваться как дополнительный способ идентификации клиента или для других нужд.

В этом разделе:
-   [Добавление и настройка дополнительного поля](#create_or_edit)
-   [Удаление дополнительного поля](#delete)

## Добавление и настройка дополнительного поля {#create_or_edit}

Чтобы добавить и настроить дополнительное поле:
1.  На главной странице или на странице «Места» выберите место и перейдите на страницу «Очереди» → выбранная очередь → «Параметры» → «Редактировать» → группа настроек «Дополнительные поля».
    :step_result: откроется страница настройки дополнительных полей.
    <ImageContainer src={"/images/lines_management/line_settings/additional_fields/additional_fields_settings.png"} description={"Группа «Дополнительные поля» в настройках очереди"}/>
2.  Нажмите «Добавить дополнительное поле».
    :step_result: в списке появится новое полею
    <ImageContainer src={"/images/lines_management/line_settings/additional_fields/additional_fields_settings_creation.png"} description={"Создание дополнительного поля"}/>
3.  Заполните основные обязательные поля:
    1.  «Название дополнительного поля» — уникальное для очереди название поля. Это название будет отображаться в интерфейсе административной панели при добавлении позиции или клиенту при записи в очередь, например, через портал бронирований.
    2.  «Краткое название» — уникальное название, которое используется в импортируемых файлах.
4.  Настройте дополнительное поле:
    1.  Нажмите на «<ExpandMore/>».
        :step_result: раскроется список настроек дополнительного поля
        <ImageContainer src={"/images/lines_management/line_settings/additional_fields/additional_fields_settings_creation_advanced.png"} description={"Настройки дополнительного поля"}/>
    2.  В поле «Описание» укажите текст, который будет отображаться как подсказка внутри поля.
    3.  В поле «Тип значения» выберите тип дополнительного поля:
        -   «Строка» — строка, тип по умолчанию.
        -   «Число» — число.
        -   «Логический флаг» — переключатель, имеет два значения: true и false.
        -   «Дата» — формат даты. При заполнении поля будет предложено использовать календарь для выбора дня, либо самостоятельно заполнение значения в формате «dd.mm.yyyy».
        -   «Выпадающий список» — список из нескольких значений.
    4.  Чтобы дополнительное поле было обязательным для заполнения, включите опцию «Обязательное поле». Для типа «Логический флаг» эта опция отсутствует, так как у такого поля значение по умолчанию равно false.
    5.  Чтобы реализовать возможность предзаполнения данных в дополнительное поле через QR-код, включите опцию «Предзаполнять при сканировании QR-кода» и в появившихся полях укажите регулярное выражение и номер группы.
        :::tip
        Для использования этой опции необходимо правильно заполнять регулярные выражения.
        Пример. В QR-коде зашифровано следующее:
        ```
        id:TS-84321854
        repeat:true
        info:anythingelse
        ```
        Чтобы для дополнительного поля «Идентификатор» извлекать идентификаторы, начинающиеся на «TS-»,
        только после «id:» в QR-коде, необходимо использовать следующее регулярное выражение:
        `(?<=id:)TS-\d+(?=\b)`
        :::
    6.  Укажите дополнительные настройки для типа поля «Строка»:
        -   «Проверять уникальность значения» — если опция включена, то в очередь будет нельзя добавить позицию со значением дополнительного поля, как у другой позиции в очереди.
        -   «Минимальная длина» и «Максимальная длина» — ограничение длины значения поля.
        -   «Регулярное выражение для проверки значения» — регулярное выражение, соответствие которому будет проверяться значение поля. Если значение неверное, то продолжить доабвление позиции или запись в очередь будет невозможно.
        -   «Сообщение при несоответствии значения регулярному выражению» — текст под полем в качестве подсказки при неверном значении поля.
    7.  Укажите список значений для типа поля «Выпадающий список»:
        1.  Нажмите «Добавить значение».
        2.  Укажите название в поле «Значение».
        3.  Укажите код значения.
        4.  Чтобы удалить значение, нажмите «<Delete/>».
5.  Добавьте аналогичным образом все дополнительные поля.
6.  Нажмите «Сохранить».
    :step_result: внизу страницы появится всплывающее уведомление «Очередь успешно обновлена».


## Удаление дополнительного поля {#delete}

Чтобы удалить неиспольуемое дополнительное поле:
1.  На главной странице или на странице «Места» выберите место и перейдите на страницу «Очереди» → выбранная очередь → «Параметры» → «Редактировать» → группа настроек «Дополнительные поля».
    :step_result: откроется страница настройки дополнительных полей.
    <ImageContainer src={"/images/lines_management/line_settings/additional_fields/additional_fields_settings.png"} description={"Группа «Дополнительные поля» в настройках очереди"}/>
2.  Выберите из списка нужное дополнительное поле и справа от него нажмите «<ExpandMore/>».
    :step_result: раскроется список настроек дополнительного поля
    <ImageContainer src={"/images/lines_management/line_settings/additional_fields/additional_fields_settings_creation_advanced.png"} description={"Настройки дополнительного поля"}/>
3.  Нажмите «Удалить».
4.  После удаления всех неиспользуемых дополнительных полей нажмите «Сохранить».
    :step_result: внизу страницы появится всплывающее уведомление «Очередь успешно обновлена».
