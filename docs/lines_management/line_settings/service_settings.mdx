# Настройка, добавление и удаление услуг в очереди {#service_settings}

import {ExpandMore} from "@mui/icons-material";
import ImageContainer from "../../../src/components/ImageContainer";

Для работы очереди нужно настроить услуги, которые клиент может получить, записываясь в очередь. Услуги нужны для определения времени обслуживания и причины обращения клиентов в очередь.

Услуги используются при настройке точек обслуживания, чтобы каждая из них получала позиции с указанными услугами.

Название услуги отображается у клиента при записи любыми способами.

В этом разделе:
-   [Настройка услуг в очереди](#configure_services)
-   [Добавление услуги в очереди](#add_service)
-   [Удаление услуги в очереди](#delete_service)

## Настройка услуг в очереди {#configure_services}

Чтобы настроить услуги в очереди:
1.  На главной странице или на странице «Места» выберите место и перейдите на страницу «Очереди» → выбранная очередь → «Параметры» → «Редактировать» → группа настроек «Услуги».
    :step_result: откроется страница настройки услуг.
    <ImageContainer src={"/images/lines_management/line_settings/services_settings/services_settings_page.png"} description={"Страница настройки услуг с тремя добавленными услугами"}/>
2.  Выберите один из типов отображения услуг:
    1.  «Обычный список» — стандартный способ отображения услуг.
    2.  «Нет» — отключает функционал услуг в очереди. При добавлении позиции нужно вручную указывать продолжительность обслуживания.
3.  Включите «Множественный выбор услуг», если нужна возможность выбора клиентом нескольких услуг в одной позиции.
    :::note
    При выборе нескольких услуг их время обслуживания будет суммироваться.
    :::
4.  Добавьте услуги в очередь, подробнее см. раздел [Добавление услуги в очереди](#add_service).
5.  Нажмите «Сохранить».
    :step_result: внизу страницы появится всплывающее уведомление «Очередь успешно обновлена».


## Добавление услуги в очереди {#add_service}

Чтобы добавить услугу и настроить ее параметры в очереди:
1.  На главной странице или на странице «Места» выберите место и перейдите на страницу «Очереди» → выбранная очередь → «Параметры» → «Редактировать» → группа настроек «Услуги».
    :step_result: откроется страница настройки услуг.
    <ImageContainer src={"/images/lines_management/line_settings/services_settings/services_settings_page.png"} description={"Страница настройки услуг с тремя добавленными услугами"}/>
2.  Нажмите «Добавить услугу».
    :step_result: появится поле для заполнения данных о новой услуге.
    <ImageContainer src={"/images/lines_management/line_settings/services_settings/create_service.png"} description={"Поля для заполнения данных о новой услуге"}/>
3.  Заполните основную информацию об услуге:
    1.  «Ключ» — краткое название, состоящее из нескольких букв на латинице. Нужен для корректной работы импорта и экспорта настроек очереди.
    2.  «Название услуги» — название для услуги, которое будет отображаться в интерфейсе ВнеОчереди и у клиентов при записи в очередь.
        :::note
        Максимальное количество символов в названии услуги — 100.
        :::
    3.  Продолжительность оказания услуги — длительность услуги в секундах.
4.  Настройте дополнительные параметры услуги:
    1.  Нажмите «<ExpandMore/>», чтобы раскрыть дополнительные параметры.
        :step_result: раскроется список дополнительных параметров.
        <ImageContainer src={"/images/lines_management/line_settings/services_settings/create_service_additional.png"} description={"Дополнительные параметры услуги"}/>
    2.  «Описание» — описание услуги, которое отображается клиенту при записи.
    3.  «Ссылка на документацию» — при заполнении в мониторинге очереди или точки обслуживания в названии услуги у позиции появится значок «![](/images/icons/ic14_question_mark_filled.svg)», при нажатии которого произойдет переход по указанной ссылке.
        <ImageContainer src={"/images/lines_management/line_settings/services_settings/position_selected_service.png"} description={"Название услуги в позиции со ссылкой на документацию"}/>
    4.  Укажите задержку между обслуживанием позиций в мультистейдж очереди в секундах. Перед началом обслуживания следующей позиции включится задержка на указанное количество секунд.
        :::info
        Этот параметр доступен только если алгоритм расчета очереди — «Маршрутная очередь». Подробнее о настройке основных параметров очереди см. в разделе [Настройка основных параметров очереди](main_settings.mdx).
        :::
    5.  Включите опцию «Техническая услуга», если она будет использоваться только для перерывов, технических обслуживаний, приостановок точки обслуживания. Техническая услуга не будет доступна для записи клиентам.
    6.  Чтобы позиция с текущей настраиваемой услугой после добавления автоматически становилась приоритетной, включите опцию «Услуга с приоритетным обслуживанием».
    7.  Включите опцию «Услуга с прогрессивным бронированием», если нужно ежедневное ограничение объема доступной предварительной записи с этой услугой.
5.  Настройте опции в дополнительных параметрах услуги, связанные с количеством участников и временем их обслуживания:
    1.  Включите опцию «Задать текст для поля "Количество участников"», чтобы изменить текст для поля «Количество участников» на свой.
        :step_result: при включении опции появится поле для ввода текста.
        <ImageContainer src={"/images/lines_management/line_settings/services_settings/custom_label_amount_persons.png"} description={"Поле для ввода текста для поля «Количество участников»"}/>
        <ImageContainer src={"/images/lines_management/line_settings/services_settings/custom_label_when_add_position.png"} description={"Пример изменения текста для поля «Количество участников» при добавлении позиции в очередь"}/>
    2.  Включите опцию «Задать максимальное значение поля "Количество участников"», чтобы указать максимальное количество участников только для текущей услуги.
        :::info
        Эта настройка влияет на возможность указать количество с другим максимальным значением только при добавлении позции через административную панель. Клиенты, записываясь в очередь, могут указать максимальное количество участников, указанное в основных параметрах очереди, подробнее см. в разделе [Настройка основных параметров очереди](main_settings.mdx).
        :::
    3.  Включите опцию «Задать значения времени обслуживания в соответствии с количеством участников», чтобы назначить время обслуживания в зависимости от количества участников.
        :step_result: появятся поля для указания времени обслуживания для каждого количества участников вплоть до максимального.
        <ImageContainer src={"/images/lines_management/line_settings/services_settings/custom_duration.png"} description={"Настройка времени обслуживания в соответствии с количеством участников"}/>
    :::info
    Опции, связанные с настройкой количества участников для услуги, доступны только если в основных параметрах очереди включена опция «Запрашивать количество человек», подробнее см. в разделе [Настройка основных параметров очереди](main_settings.mdx).
    :::
6.  Нажмите «Сохранить».
    :step_result: внизу страницы появится всплывающее уведомление «Очередь успешно обновлена».


## Удаление услуги в очереди {#delete_service}

Чтобы удалить неиспользуемую услугу из очереди:
1.  На главной странице или на странице «Места» выберите место и перейдите на страницу «Очереди» → выбранная очередь → «Параметры» → «Редактировать» → группа настроек «Услуги».
    :step_result: откроется страница настройки услуг.
    <ImageContainer src={"/images/lines_management/line_settings/services_settings/services_settings_page.png"} description={"Страница настройки услуг с тремя добавленными услугами"}/>
2.  Нажмите «<ExpandMore/>» у услуги, которую нужно удалить, чтобы раскрыть дополнительные параметры.
    :step_result: раскроется список дополнительных параметров.
    <ImageContainer src={"/images/lines_management/line_settings/services_settings/create_service_additional.png"} description={"Дополнительные параметры услуги"}/>
3.  Нажмите «Удалить».
    :step_result: услуга пропадет из списка.
4.  Нажмите «Сохранить».
    :step_result: внизу страницы появится всплывающее уведомление «Очередь успешно обновлена».
