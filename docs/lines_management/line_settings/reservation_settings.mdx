# Настройка параметров предварительной записи {#reservation_settings}

В Системе предусмотрена возможность записи клиентов на определенную дату, для этого нужно настроить параметры предварительной записи в настройках очереди.

Для настройки предварительной записи:
1.  На главной странице выберите место и перейдите на страницу «Очереди» → выбранная очередь → «Параметры» → «Редактировать» → группа настроек «Бронирование на будущие дни».
    <ImageContainer src={"/images/lines_management/line_settings/reservation_settings.png"} description={"Группа «Бронирования на будущие дни» в настройках очереди"}/>
2.  Включите «Разрешать бронирование на будущие дни», чтобы клиент мог создать предварительную запись.
3.  Укажите «Количество доступных дней бронирования для пользователя» — максимальное количество дней, которые будут предложены клиенту.
4.  Чтобы ограничить количество слотов для записи на будущие дни:
    1.  Включите «Прогрессивное бронирование».
    2.  Укажите «Порог предоставления всех встреч» — количество дней после текущего, в которых для предварительной записи будет доступно все время дня.
    :::info
    При включении функции прогрессивного бронирования будет открываться для записи не весь день целиком, а только его часть. Чем ближе текущая дата к этому дню, тем больше времени будет доступно для предварительной записи.
    :::
5.  Включите «Разрешить несколько встреч в один день» для безлимитного количества записей на один день от одного клиента.
6.  Включите «Разрешить наслоение записей одного участника», чтобы клиент мог создать предварительную запись на время, которое пересекается с временем другой записи.
7.  Включите «Видимость в веб-портале для встреч», чтобы очередь отображалась в веб-портале.
8.  Чтобы ограничить время для создания предварительных записей:
    1.  Включите «Бронирования осуществляются в установленные часы».
    2.  Укажите начало и конец периода, во время которого клиентам можно создавать предварительную запись.
9.  При необходимости сделайте обязательными для заполнения поля при создании предварительной записи:
    1.  Включите «Электронный адрес и телефон обязательный».
    2.  Включите «Фамилия обязательна».
10. При необходимости включите:
    1.  «Проверять электронный адрес кодом подтверждения» — клиенту на почту придет шестизначный код подтверждения, который нужно будет ввести перед созданием предварительной записи.
    2.  «Прикреплять файл ics к электронному письму с подтверждением» — после создания записи клиенту на почту придет файл формата ICS для загрузки в календарь.
11. Включите «Бронирования на текущий день в веб портале», чтобы клиент мог создать предварительную запись на текущий день.
12. Укажите «Ограничения предложений на будущие дни для администратора» для возможности переноса записи клиента в случае непредвиденных обстоятельств.
13. Настройте напоминания клиенту:
    1.  Через электронную почту:
        1.  Включите «Отправлять напоминание о бронировании по электронной почте».
        2.  Укажите «Отправлять напоминание за» в днях.
    2.  Через SMS:
        1.  Включите «Отправлять SMS сообщение с напоминанием о бронировании».
        2.  Укажите «Отправлять напоминание за» в днях.
14. Включите опцию «Запрашивать комментарии от пользователя», чтобы клиент мог указывать комментарий при создании предварительной записи.
15. Включите опцию «Показывать временные интервалы на сегодня при записи через административную панель», чтобы сотрудник мог создать предварительную запись для клиента на сегодняшний день в Системе через раздел «Бронирования».
16. Чтобы блокировать временной интервал после отмены предварительной записи:
    1.  Включите «Временно заблокировать временной интервал после его освобождения».
    2.  Укажите минимальное и максимальное время блокировки. Время блокировки будет выбрано случайно между этим интервалом.
17. Укажите «Email для копий с подтверждениями бронирований» через запятую, чтобы письма с подтверждениями дублировались на дополнительные почты.
18. Нажмите «Сохранить».
