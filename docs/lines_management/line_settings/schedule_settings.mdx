# Настройка времени открытия очереди {#schedule_settings}

Время открытия очереди определяет график работы по умолчанию для точек обслуживания и определяет дни и время, в которые работает очередь.

Для настройки времени открытия очереди:
1.  На главной странице выберите место и перейдите на страницу «Очереди» → выбранная очередь → «Параметры» → «Редактировать» → группа настроек «Время открытия».
    <ImageContainer src={"/images/lines_management/line_settings/schedule_settings.png"} description={"Группа «Время открытия» в настройках очереди"}/>
2.  Включите «Круглосуточно» для работы очереди 24/7.
3.  Включите «Очередь с перерывом» и укажите начало и окончание перерыва, чтобы добавить в очередь перерывы, во время которых работа точек обслуживания приостанавливается.
4.  Включите «Автоматически закрывать точки обслуживания ночью» для закрытия точек обслуживания в 00:00.
5.  Настройте график работы:
    1.  Укажите «Время открытия».
    2.  Укажите «Время закрытия».
6.  Нажмите «Сохранить».
