# Настройка информационных тегов очереди {#additional_info}

В Системе можно прикреплять к позициям в очереди теги дополнительной информации о клиенте. Выбирать теги можно только из списка, который формируется в настройках очереди.

Чтобы настроить информационные тэги:
1.  На главной странице выберите место и перейдите на страницу «Очереди» → выбранная очередь → «Параметры» → «Редактировать» → группа настроек «Информационные тэги».
    <ImageContainer src={"/images/lines_management/line_settings/info_tags_settings.png"} description={"Группа «Информационные тэги» в настройках очереди"}/>
2.  Включите дополнительную информацию.
    <ImageContainer src={"/images/lines_management/line_settings/info_tags_settings_enabled.png"} description={"Включенные информационные теги"}/>
3.  Введите названия тегов в поле «Тэги разделенные ;» через точку с запятой.
4.  При необходимости включите «Несколько информационных тэгов могут быть указаны одновременно».
5.  Нажмите «Сохранить».
