# Настройка тегов очереди {#tags_creation}

import ImageContainer from "../../src/components/ImageContainer";
import {Add, Delete} from "@mui/icons-material";

Теги используются для группировки очередей разных мест в одну категорию для их отображения в мобильном приложении. Пользователю мобильного приложения отображаются все очереди не списком, а группами, к которым был назначен тег.

На примере ниже «МФЦ г. Екатеринбурга для физических лиц» — это тег, который был назначен к очередям во время их настройки.

<ImageContainer src={"/images/lines_management/mobile_app_lines_page.jpg"} description={"Список очередей в мобильном приложении"}/>

Чтобы создать и настроить тег:
1.  На главной странице перейдите на страницу «Теги».
2.  Нажмите «<Add/>».
    <ImageContainer src={"/images/lines_management/tag_creation_page.png"} description={"Страница создания тега"}/>
3.  Введите название тега в поле «Тег».
4.  Добавьте очереди в тег одним из способов:
    1.  Нажмите «Сохранить» для создания тега без очередей и добавьте тег к очереди при ее настройке, подробнее см. в разделе [Настройка основных параметров](line_settings/main_settings.mdx).
    2.  Выберите все нужные места и очереди:
        1.  Нажмите «Добавить место».
        2.  Выберите место и нажмите «<Add/>».
            <ImageContainer src={"/images/lines_management/tag_creation_page_added_shop.png"} description={"Добавление места в тег"}/>
        3.  Нажмите «Добавить очередь».
        4.  Выберите очередь места и нажмите «<Add/>».
            <ImageContainer src={"/images/lines_management/tag_creation_page_added_line.png"} description={"Добавление очереди места в тег"}/>
        5.  Добавьте все места и очереди подобным образом.
            :::tip
            Для удаления места или очереди используйте «<Delete/>».
            :::
5.  Нажмите «Сохранить».
    :step_result: откроется страница «Теги» с добавленным в список тегом.
