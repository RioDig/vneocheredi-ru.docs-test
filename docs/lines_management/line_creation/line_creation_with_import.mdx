# Создание очереди с импортированными настройками {#line_creation_with_import}

При создании очереди для ускорения процесса настройки можно воспользоваться функцией импорта настроек. Для этого:
1.  [Экспортируйте настройки нужной очереди](#export_settings)
2.  [Импортируйте настройки при создании очереди](#import_settings)

## Экспорт настроек очереди {#export_settings}

:::tip
Если файл настроек очереди в формате JSON уже создан, пропустите этот шаг.
:::

Перед импортом настроек при создании новой очереди нужно получить файл формата JSON. Для этого:
1.  На главной странице выберите место и перейдите на страницу «Очереди» → выбранная очередь → «Параметры» → «Редактировать» → группа настроек «Расширенные настройки».
    <ImageContainer src={"/images/lines_management/line_settings/advanced_settings.png"} description={"Группа «Расширенные настройки» в настройках очереди"}/>
2.  Нажмите «Экспортировать» в поле «Экспортировать как шаблон».
    :step_result: будет скачан файл формата JSON со списком настроек очереди.


## Импорт настроек при создании очереди {#import_settings}

При наличии файла формата JSON со списком настроек очереди его можно использовать при создании очереди.

Чтобы создать очередь с импортом настроек из файла:
1.  На главной странице откройте выбранное место и перейдите на страницу «Очереди».
    <ImageContainer src={"/images/lines_management/lines_page.png"} description={"Интерфейс страницы «Очереди»"}/>
2.  Нажмите «<Add/>».
    <ImageContainer src={"/images/lines_management/line_create.png"} description={"Группа «Параметры» в настройках очереди"}/>
3.  Введите название очереди в поле «Название».
4.  Нажмите «Импортировать параметры из файла» под списком групп настроек очереди.
5.  В открывшемся окне выберите файл формата JSON с настройками очереди.
    :step_result: настройки из файла применятся к новой очереди.
6.  При необходимости измените настройки очереди.
    :::info
    О настройке очереди см. в разделе [Настройка очереди](../line_settings/line_settings.mdx).
    :::
7.  Нажмите «Сохранить».
    :step_result: откроется страница «Очереди», на которой появится созданная очередь.
