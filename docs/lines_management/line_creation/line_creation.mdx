# Создание очереди {#line_creation}

Для создания очереди необходимо указать только ее название и состав услуг. Этого достаточно, чтобы начать пользоваться ею. Ниже приведены инструкции для создания очереди разными способами.

:::info
Для дальнейшей настройки очереди см. раздел [Настройка очереди](../line_settings/line_settings.mdx).
:::

Создать очередь можно следующими способами:
-   С добавленными услугами — см. в разделе [Создание очереди с услугами](line_creation_with_services.mdx).
-   Без услуг — см. раздел [Создание очереди без услуг](line_creation_without_services.mdx).
-   Через импорт настроек другой очереди — см. в разделе [Создание очереди с импортированными настройками](line_creation_with_import.mdx).
