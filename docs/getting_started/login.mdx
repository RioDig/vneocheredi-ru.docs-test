# Вход в административную панель ВнеОчереди {#login}

import ImageContainer from "../../src/components/ImageContainer";


В системе ВнеОчереди самостоятельная регистрация пользователей не предусмотрена.
Учетные записи создаются администраторами, которые затем передают данные для входа сотрудникам,
например операторам точки обслуживания. После получения учетных данных пользователи могут авторизоваться и получить доступ к административной панели.

:::note
Если вы являетесь администратором, но у вас отсутствует учетная запись, пожалуйста, обратитесь в техническую поддержку для её создания.
:::

Чтобы войти в административную панель ВнеОчереди:
1.  Откройте административную панель ВнеОчереди.
    :step_result: откроется страница входа.
    <ImageContainer src={"/images/getting_started/login_page.png"} description={"Страница входа ВнеОчереди"}/>
2.  Введите адрес электронной почты и пароль, полученные от администратора ВнеОчереди.
3.  Нажмите «Войти».
    :step_result: откроется главная страница ВнеОчереди.
    :::caution
    Если вы входите в систему впервые, настоятельно рекомендуем сразу сменить пароль вашей учетной записи,
    подробнее см. в разделе [Изменение пароля, персональных данных и языка интерфейса своей учетной записи](../account_management/password_change.mdx).
    :::
