# Создание и настройка мест, очередей и точек обслуживания {#shop_line_checkpoint_creation}

После входа в систему ВнеОчереди администратору необходимо выполнить первоначальную настройку. Для этого требуется создать и настроить места, очереди и точки обслуживания. Обратите внимание, что ВнеОчереди использует систему ролей и прав доступа, поэтому выполнять вышеуказанные действия смогут только пользователи с определенными правами доступа, подробнее см. в разделе [Права доступа](../account_management/access_rights.mdx).

Продолжите настройку ВнеОчереди в предложенном порядке:
1.  Создайте и настройте место, подробнее см. в разделе [Создание, настройка и удаление места](../shops_management/shops_creation_and_settings.mdx).
2.  Создайте и настройте необходимые очереди:
    1.  [Создание очереди](../lines_management/line_creation/line_creation.mdx)
    2.  [Настройка очереди](../lines_management/line_settings/line_settings.mdx)
3.  Создайте и настройте точки обслуживания, подробнее см. в разделе [Управление точками обслуживания](../checkpoint_management/checkpoint_management.mdx).
