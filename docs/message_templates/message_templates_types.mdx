# Типы шаблонов сообщений {#message_templates_types}

-   [Общие](#general)
-   [Предварительные записи](#reservations)

## Общие {#general}

| Название шаблона сообщения                     | Описание шаблона сообщения                                                                                                                                                                                                                                                                                        | Пример использования \(нажмите на картинку, чтобы увеличить\)                     |
|------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------|
| Тема письма с подтверждением записи на сегодня | Отображается в теме письма клиенту при записи на сегодняшний день \(не живая очередь\)                                                                                                                                                                                                                            | <ImageContainer src={"/images/message_templates/message_templates_types/1.png"}/> |
| Тело письма с подтверждением записи на сегодня | Отображается в теле письма клиенту при записи на сегодняшний день \(не живая очередь\)                                                                                                                                                                                                                            | <ImageContainer src={"/images/message_templates/message_templates_types/2.png"}/> |
| Текст с просьбой указать количество участников | Отображается в мобильном приложении/портале бронирований/терминале типа «Веб-терминал» на этапе выбора количества участников. Также отображается на экране текущей встречи в верхней части в мобильном приложении и на странице «Бронирования» в административной панели во время создания предварительной записи | <ImageContainer src={"/images/message_templates/message_templates_types/3.jpg"}/> |
| Советы после присоединения к очереди           | Отображается в мобильном приложении после успешной записи в очередь                                                                                                                                                                                                                                               | <ImageContainer src={"/images/message_templates/message_templates_types/4.jpg"}/> |
| Текст ожидания в ноль минут                    | Отображается в мобильном приложении на экране текущей встречи, когда примерное время ожидания истекло \(равно 0 минут, после времени предварительной записи, когда очередь опаздывает\)                                                                                                                           | <ImageContainer src={"/images/message_templates/message_templates_types/5.jpg"}/> |
| Поздравление с окончанием обслуживания         | Отображается в мобильном приложении после окончания обслуживания \(или если позиция была удалена\)                                                                                                                                                                                                                | <ImageContainer src={"/images/message_templates/message_templates_types/6.jpg"}/> |
| Уведомление о перегруженности очереди          | Отображается при нажатии на очередь в **мобильном приложении** на очередь, которая сейчас работает, но все слоты для записи уже заняты                                                                                                                                                                            | <ImageContainer src={"/images/message_templates/message_templates_types/7.jpg"}/> |
| Уведомление о том, что очередь закрыта         | Отображается при нажатии на очередь в **терминале типа «Веб-терминал»** или в **мобильном приложении** на очередь, которая сейчас не работает                                                                                                                                                                     | <ImageContainer src={"/images/message_templates/message_templates_types/8.jpg"}/> |

## Предварительные записи {#reservations}

| Название шаблона сообщения                          | Описание шаблона сообщения                                                                                                                                                                                                                                                                   | Пример использования \(нажмите на картинку, чтобы увеличить\)                      |
|-----------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------|
| Резюме предварительной записи для печати            | Отображается в версии для печати при нажатии сотрудником на кнопку «Печать» у карточки предварительной записи на странице «Бронирования»                                                                                                                                                     | <ImageContainer src={"/images/message_templates/message_templates_types/9.jpg"}/>  |
| Verification code email subject                     | Отображается в теме письма клиенту после указания персональных данных клиента в мобильном приложении или на портале бронирований. Работает только если в настройках очереди в группе настроек «Бронирования на будущие дни» включена опция «Проверять электронный адрес кодом подтверждения» | <ImageContainer src={"/images/message_templates/message_templates_types/10.jpg"}/> |
| Verification code email body                        | Отображается в теле письма клиенту после указания персональных данных клиента в мобильном приложении или на портале бронирований. Работает только если в настройках очереди в группе настроек «Бронирования на будущие дни» включена опция «Проверять электронный адрес кодом подтверждения» | <ImageContainer src={"/images/message_templates/message_templates_types/11.jpg"}/> |
| Резюме предварительной записи                       | Отображается у клиента на финальной странице после записи через портал бронирований                                                                                                                                                                                                          | <ImageContainer src={"/images/message_templates/message_templates_types/12.jpg"}/> |
| Тема письма с подтверждением предварительной записи | Отображается в теме письма клиенту после создания предварительной записи \(бронирования\)                                                                                                                                                                                                    | <ImageContainer src={"/images/message_templates/message_templates_types/13.jpg"}/> |
| Тело письма с подтверждением предварительной записи | Отображается в теле письма клиенту после создания предварительной записи \(бронирования\)                                                                                                                                                                                                    | <ImageContainer src={"/images/message_templates/message_templates_types/14.jpg"}/> |
| Тема письма с напоминанием о предварительной записи | Отображается в теме письма клиенту за N дней до встречи. Работает только если в настройках очереди в группе настроек «Бронирования на будущие дни» включена опция «Отправлять напоминание о бронировании по электронной почте» и указано «Отправлять напоминание за» равное N                |                                                                                    |
| Тело письма с напоминанием о предварительной записи | Отображается в теле письма клиенту за N дней до встречи. Работает только если в настройках очереди в группе настроек «Бронирования на будущие дни» включена опция «Отправлять напоминание о бронировании по электронной почте» и указано «Отправлять напоминание за» равное N                |                                                                                    |
| Тема письма с подтверждением предварительной записи | Отображается в теме письма клиенту после нажатия сотрудником на кнопку «Подтвердить» у карточки предварительной записи на странице «Бронирования»                                                                                                                                            | <ImageContainer src={"/images/message_templates/message_templates_types/15.jpg"}/> |
| Тело письма с подтверждением предварительной записи | Отображается в теле письма клиенту после нажатия сотрудником на кнопку «Подтвердить» у карточки предварительной записи на странице «Бронирования»                                                                                                                                            | <ImageContainer src={"/images/message_templates/message_templates_types/16.jpg"}/> |
| Резюме отмены предварительной записи                | Отображается на странице портала бронирований после нажатия на кнопку «Отменить запись»                                                                                                                                                                                                      | <ImageContainer src={"/images/message_templates/message_templates_types/17.jpg"}/> |
| Тема письма об отмене предварительной записи        | Отображается в теме письма клиенту после удаления предварительной записи \(бронирования\) через административную панель или если клиент самостоятельно отменил запись                                                                                                                        | <ImageContainer src={"/images/message_templates/message_templates_types/18.jpg"}/> |
| Тело письма об отмене предварительной записи        | Отображается в теле письма клиенту после удаления предварительной записи \(бронирования\) через административную панель или если клиент самостоятельно отменил запись. Также отображается в мобильном приложении во всплывающем окне после отмены клиентом предварительной записи            | <ImageContainer src={"/images/message_templates/message_templates_types/19.jpg"}/> |
| Адрес предварительной записи                        | Отображается в файле формата ICS в поле для указания места/адреса, который прикрепляется к письму при создании, подтверждении или отмене предварительной записи                                                                                                                              | <ImageContainer src={"/images/message_templates/message_templates_types/20.jpg"}/> |

## Другие {#other}

| Название шаблона сообщения                             | Описание шаблона сообщения                                                                                                                                                    | Пример использования \(нажмите на картинку, чтобы увеличить\)                      |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------|
| Голосовой вызов                                        | Воспроизводится голосом на терминале типа «Экран вызова», если у терминала установлен режим «Голосовой вызов» или «Смешанный режим»                                           |                                                                                    |
| Бумажный билетик                                       | Отображается на талоне при печати через терминал, либо если сотрудник нажал на кнопку «Печать» в карточке позиции                                                             | <ImageContainer src={"/images/message_templates/message_templates_types/25.jpg"}/> |
| Поздравления с присоединением к очереди через терминал | Отображается на финальном экране терминала после записи через него. Работает если в настройках терминала включен параметр «Показывать финальный экран с информацией о записи» | <ImageContainer src={"/images/message_templates/message_templates_types/26.jpg"}/> |
