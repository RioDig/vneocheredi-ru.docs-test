# Изменение пароля, персональных данных и языка интерфейса своей учетной записи {#password_change}

import ImageContainer from "../../src/components/ImageContainer";

Зарегистрированные и авторизованные пользователи могут редактировать свою учетную запись: изменять имя, фамилию, адрес электронной почты, пароль, а также язык интерфейса в административной панели ВнеОчереди.

Чтобы изменить персональные данные своей учетной записи и язык интерфейса административной панели:

1.  В верхнем правом углу страницы нажмите на свой аватар или имя.
    :step_result:появится всплывающее окно с именем, адресом электронной почты и кнопками «Настройки» и «Выход».
    <ImageContainer src={"/images/account_management/account_modal.png"} description={""}/>
2.  Нажмите «Настройки».
    :step_result: откроется страница параметров вашей учетной записи.
    <ImageContainer src={"/images/account_management/account_settings.png"} description={"Страница параметров учетной записи"}/>
3.  Измените персональные данные учетной записи: имя, фамилия, адрес электронной почты.
4.  Чтобы изменить язык интерфейса административной панели ВнеОчереди, нажмите на поле «Язык интерфейса» и выберите один из предложенных языков.
5.  Чтобы изменить пароль учетной записи:
    1.  Нажмите на блок «Изменить пароль».
        :step_result: откроется форма изменения пароля.
        <ImageContainer src={"/images/account_management/account_password_change.png"} description={"Форма изменения пароля"}/>
    2.  В появившихся полях введите текущий пароль, новый пароль и подтверждение пароля.
        :::caution
        Пароль должен состоять из 8 символов, иметь заглавную букву, цифру и специальный символ.
        :::
6.  Нажмите «Сохранить».
    :step_result: внизу страницы появится всплывающее уведомление «Пользователь успешно обновлен».
