# Права доступа {#access_rights}


При создании или редактировании пользователя можно назначать ему отдельные права доступа. На данный момент представлено две группы прав доступа:

-   [Права на действия с позицией](#position_actions)
-   [Права на доступ к разделам](#pages_access)

:::note
Подробнее о создании и настройке пользователя см. в разделе [Создание и настройка учетных записей](account_creation_and_settings.mdx).
:::
