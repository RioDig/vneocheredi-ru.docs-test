# Редактирование и удаление учетных записей {#account_edit_or_delete}

import ImageContainer from "../../src/components/ImageContainer";
import {FilterList, RemoveCircleOutlineRounded} from "@mui/icons-material";

:::info
Редактировать и удалять учетные записи других пользователей можно только пользователям с правом доступа
«Доступ к разделу "Пользователи"». Подробнее о правах доступа см. в разделе [Управление учетными записями](account_management.mdx).

Для редактирования своей учетной записи перейдите в раздел [Изменение пароля, персональных данных и языка интерфейса своей учетной записи](password_change.mdx).
:::

В этом разделе:

-   [Редактирование учетной записи](#edit)
-   [Удаление учетной записи](#delete)

## Редактирование учетной записи {#edit}

Чтобы изменить параметры учетной записи пользователя или его права доступа:
1.  С главной страницы перейдите на страницу «Пользователи».
    <ImageContainer src={"/images/account_management/accounts_page.png"} description={"Интерфейс страницы «Пользователи»"}/>
2.  Выберите из списка нужного пользователя и нажмите на него.
    :::tip
    Для экономии времени используйте поиск и фильтры:
    -   Строка поиска — для поиска по имени, фамилии, адресу электронной почты, описанию пользователя.
    -   Фильтры «<FilterList/>» — для фильтрации по наличию доступа к местам, по типу пользователя, по статусу «Неактивен».
    :::
    :step_result:откроется страница параметров учетной записи выбранного пользователя.
    <ImageContainer src={"/images/account_management/account_settings_page.png"} description={"Страница параметров пользователя"}/>
3.  При необходимости измените данные пользователя: имя, фамилия, адрес электронной почты, язык интерфейса, описание.
4.  В случае, если пользователь забыл пароль и не может его восстановить, измените пароль:
    1.  Нажмите на блок «Изменить пароль».
    2.  Введите новый пароль.
        :::caution
        Пароль должен состоять из 8 символов, иметь заглавную букву, цифру и специальный символ.
        :::
5.  При необходимости измените права и доступ пользователю:
    1.  Нажмите на блок «Права и доступ».
        :step_result:откроется блок для настройки прав и доступа.
        <ImageContainer src={"/images/account_management/account_create_access_and_rights.png"} description={"Блок «Права и доступ» в настройках пользователя"}/>
    2.  «Активен» — деактивировать или активировать учетную запись пользователя. Деактивированные пользователи не смогут авторизоваться во ВнеОчереди.
    3.  «Системный администратор» — опция, позволяющая сделать пользователя администратором всей системы.
        :::danger
        Будьте осторожны при включении этой опции. Системный администратор имеет доступ ко всем функциям.
        :::
        :::info
        Включение опции «Системный администратор» доступна только другим системным администраторам.
        :::
    4.  Выберите тип пользователя:
        -   «Оператор» — для настройки пользователя как оператора очереди. При отсутствии других прав
            доступа имеет доступ только к просмотру очередей и точкам обслуживания.
        -   «Администратор» — для настройки пользователя как любую другую роль. При отсутствии других прав
            доступа обладает правами Оператора и имеет доступ к настройке графика сотрудников, выходных дней, маячков.
        :::tip
        Тип пользователя определяет возможность настройки модулей места. Тип «Оператор» подходит, если
        пользователь будет работать только с позициями клиентов и не будет администрировать систему.
        В остальных случаях лучше использовать «Администратор».
        :::
    5.  Установите права доступа в блоках «Действия с позицией» и «Доступ к разделам». Подробнее о правах см. в разделе [Права доступа](access_rights.mdx).
        :::info
        Если вы не являетесь Системным администратором, то можете предоставлять только те права, которые уже назначены вашему аккаунту.
        :::
6.  При необходимости измените список мест, к которым имеет доступ пользователь:
    1.  Нажмите на блок «Места».
    2.  Выберите место из списка и нажмите «Добавить место». Чтобы удалить место из списка, нажмите «<RemoveCircleOutlineRounded/>» справа от него.
7.  Нажмите «Сохранить».
    :step_result: внизу страницы появится всплывающее уведомление «Пользователь успешно обновлен».


## Удаление учетной записи {#delete}

Чтобы удалить учетную запись пользователя:
1.  С главной страницы перейдите на страницу «Пользователи».
    <ImageContainer src={"/images/account_management/accounts_page.png"} description={"Интерфейс страницы «Пользователи»"}/>
2.  Выберите из списка нужного пользователя и нажмите на него.
    :::tip
    Вы можете воспользоваться поиском и фильтрами для удобства:
    -   Строка поиска — для поиска по имени, фамилии, адресу электронной почты, описанию пользователя.
    -   Фильтры «![](/images/icons/ic24_filter.svg)» — для фильтрации по наличию доступа к местам, по типу пользователя, по статусу «Неактивен».
    :::
    :step_result: откроется страница параметров учетной записи выбранного пользователя.
    <ImageContainer src={"/images/account_management/account_settings_page.png"} description={"Страница параметров пользователя"}/>
3.  Внизу страницы нажмите «Удалить» и подтвердите действие.
    :step_result: внизу страницы появится всплывающее уведомление «Пользователь успешно удален».
