# Поиск записи в журнале {#getting_log}

Система хранит историю всех записей клиентов в очередь, технических перерывов и предварительных записей по каждому месту и очереди. В каждой записи журнала есть история действий по ней с указанием времени и даты, подробнее об информации в записи журнала см. в разделе [Просмотр записи в журнале](view_log_entry.mdx).

:::info
Чтобы просматривать журнал, пользователь должен обладать правом «Доступ к разделу "Выгрузка данных" и "Журнал"». Подробнее о правах пользователей см. в разделе [Права доступа](../account_management/access_rights.mdx).
:::

Чтобы найти запись в журнале:
1.  На главной странице перейдите на страницу «Журнал».
    :step_result: откроется страница журнала.
    <ImageContainer src={"/images/journal/journal_page.png"} description={"Страница «Журнал»"}/>
2.  Задайте условия поиска:
    -   В поле «Поиск» условиями могут быть персональные данные клиента или сотрудника, код бронирования, номер записи в журнале и т.д.
    -   В полях «Место» и «Очередь» указывается, где была запись клиента или сотрудника.
    -   В полях «Дата от» и «Дата до» указывается промежуток дат записи клиентов или сотрудников.
3.  Нажмите «Найти».
    :step_result: отобразится список записей в журнале, удовлетворяющий критериям поиска.
