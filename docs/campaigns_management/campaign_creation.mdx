# Создание рекламной кампании {#campaign_creation}

import {Add} from "@mui/icons-material";
import ImageContainer from "../../src/components/ImageContainer";

Для подключения рекламы к терминалам места или в мобильное приложение нужно создать рекламную кампанию, которая имеет срок показа рекламы, продолжительность показа, привязку к одному или нескольким местам и рекламные баннеры разного типа. Подробнее о добавлении рекламных баннеров в кампанию см. в разделе [Добавление рекламных баннеров](banner_creation.mdx).

Чтобы создать рекламную кампанию:
1.  На главной странице перейдите на страницу «Реклама».
2.  Нажмите «<Add/>».
    <ImageContainer src={"/images/campaigns_management/campaign_creation.png"} description={"Страница создания рекламной кампании"}/>
3.  Введите название рекламной кампании, оно будет отображаться на странице «Реклама».
4.  Укажите начальную и конечную даты рекламной кампании. Реклама будет активна только в этом промежутке.
5.  Укажите продолжительность показа рекламы в секундах. Этот параметр определяет продолжительность показа
    каждого рекламного баннера, если у него не указана своя продолжительность.
6.  При необходимости выберите статус рекламной кампании. Статус по умолчанию — «Ожидает активации».
7.  Выберите места, где будет показываться реклама.
8.  Нажмите «Сохранить».
    :step_result: откроется страница «Реклама» с добавленной рекламной кампанией.
9.  Добавьте рекламные баннеры, подробнее см. в разделе [Добавление рекламных баннеров](banner_creation.mdx).
