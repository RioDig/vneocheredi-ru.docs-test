## Включение аудиофиксации {#enable_recording}

Для использования функции аудиофиксации необходимо изменить параметры очереди:
1.  Перейдите на страницу «Очереди», выберите нужную очередь и перейдите в ее параметры.
    :step_result: откроется страница параметров очереди.
    <ImageContainer src={"/images/instructions/audio_recording/line_settings_main.png"} description={"Страница параметров очереди"}/>
2.  Перейдите в раздел настроек «Расширенные настройки».
    :step_result: откроется страница расширенных параметров очереди.
    <ImageContainer src={"/images/instructions/audio_recording/line_settings_advanced.png"} description={"Страница параметров очереди, группа настроек «Расширенные настройки»"}/>
3.  Выберите один из режимов в параметре «Запись голоса в мониторинге точки обслуживания»:
    1.  «Отключена» — функция аудиофиксации выключена.
    2.  «Если есть доступ к микрофону» — аудиофиксация будет включена, если пользователь даст
        разрешение на доступ к микрофону на странице мониторинга точки обслуживания. **Позиции в мониторинге точки обслуживания будут доступны в любом случае.**
    3.  «Запись обязательна» — аудиофиксация будет включена, если пользователь дал разрешение
        на доступ к микрофону. **Позиции в мониторинге точки обслуживания будут доступны только после разрешения использования микрофона в браузере.**

    :step_result: при включении аудиофиксации появится поле «Время хранения аудиозаписей».
4.  В появившемся поле «Время хранения аудиозаписей \(дней\)» укажите количество дней хранения аудиозаписи в журнале.
5.  Нажмите «Сохранить».
    :step_result: внизу страницы появится всплывающее уведомление «Очередь успешно обновлена».
6.  Перейдите на страницу «Точки обслуживания».
    :step_result: откроется страница со списком точек обслуживания очереди.
    <ImageContainer src={"/images/instructions/audio_recording/checkpoints_page.png"} description={"Страница «Точки обслуживания»"}/>
7.  Нажмите «<Edit/>», чтобы перейти в параметры точки обслуживания.
    :step_result: откроется страница параметров точки обслуживания.
    <ImageContainer src={"/images/instructions/audio_recording/checkpoint_settings_page.png"} description={"Страница параметров точки обслуживания"}/>
8.  Включите опцию «Включить аудиозапись разговоров».
9.  Нажмите «Сохранить».
    :step_result: внизу страницы появится всплывающее уведомление «Точка обслуживания успешно обновлена».
10. Повторите шаги 7-9 для всех точек обслуживания, где нужно включить функцию аудиофиксации.

## Предоставление доступа на использование микрофона в браузере {#monitoring_with_audiorecording}

После включения аудиофиксации для точки обслуживания сотруднику при первом посещении мониторинга точки обслуживания будет предложено разрешить браузеру использовать микрофон.

<ImageContainer src={"/images/instructions/audio_recording/access_request.png"} description={"Запрос разрешений на использование микрофона"}/>

Если сотрудник запретил использование микрофона или случайно пропустил запрос разрешений, то аудиофиксация не будет
работать. Чтобы решить эту проблему, предоставьте доступ к микрофону в браузере. Ниже представлены инструкции для разных браузеров:
-   [Для Google Chrome](https://support.google.com/chrome/answer/2693767?hl=ru)
-   [Для Mozilla Firefox](https://support.mozilla.org/ru/kb/upravlenie-razresheniyami-dlya-kamery-i-mikrofona-#w_ispolzovanie-meniu-toptionsorpreferences-firefox-dlia-izmeneniia-razreshenii-dlia-kamery-i-mikrofona-na-kompiutere)
-   [Для Яндекс Браузер](https://browser.yandex.ru/help/ru/personal-data-protection/do-not-follow#microphone)

## Начало аудиофиксации в мониторинге точки обслуживания {#start_audiorecording}

Аудиофиксация включается после начала обслуживания \(действие «Начать обслуживание»\), при этом в праврм нижнем углу страницы появляется индикатор записи.
<ImageContainer src={"/images/instructions/audio_recording/rec_indicator.png"} description={"Индикатор, появляющийся при начале обслуживания позиции"}/>

:::info
Если сотрудник уходит со страницы мониторинга точки обслуживания, запись разговора прекращается. При переключении обратно на страницу начинается новая запись.
:::

## Просмотр записей в журнале {#view_recordings}

Все записи разговоров, сделанные при помощи функции аудиофиксации, сохраняются в журнале для конкретной позиции. Чтобы найти и просмотреть аудиозаписи:
1.  С главной страницы перейдите на страницу «Журнал».
    :step_result: откроется страница журнала.
    <ImageContainer src={"/images/instructions/audio_recording/journal_page.png"} description={"Страница журнала"}/>
2.  Укажите критерии поиска и нажмите кнопку «Найти».
    :step_result: справа появятся записи, удовлетворяющие критериям поиска.
    <ImageContainer src={"/images/instructions/audio_recording/journal_page_found.png"} description={"Найденная запись в журнале"}/>
3.  Раскройте блок «Аудиозаписи» у найденной позиции в журнале.
    :step_result: отобразится список аудиозаписей, отсортированный по времени записи.
    <ImageContainer src={"/images/instructions/audio_recording/journal_recordings.png"} description={"Аудиозаписи, сделанные во время обслуживания позиции"}/>
4.  Чтобы скачать аудиозапись, нажмите «<MoreVert/>» → «Скачать».
5.  При необходимости нажмите «<Delete/>» и подтвердите действие, чтобы удалить аудиозапись.
